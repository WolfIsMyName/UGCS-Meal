## 定义

- T0：性价比高、味道好、位置好，顿顿吃无压力，经常会去吃

- T1：在性价比、味道、地理位置中个别项不占优势，会主动提出去吃

- T2：在性价比、味道、地理位置中多项不占优势，或一项极具劣势，一般不会主动提出去吃，但是可以接受

- T3：狗都不吃
